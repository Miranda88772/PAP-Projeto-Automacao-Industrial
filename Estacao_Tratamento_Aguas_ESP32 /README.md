# PAP – Sistema Integrado de Automação e Monitorização

## Descrição do Projeto

Este repositório reúne os códigos-fonte desenvolvidos no âmbito da Prova de Aptidão Profissional (PAP) do curso Técnico de Eletrónica, Automação e Computadores.

O projeto consiste num sistema integrado de automação e monitorização composto por três módulos principais:

### 1. Estação de Tratamento de Águas (ETA)

Sistema responsável pela monitorização e controlo de diversos parâmetros da água, incluindo nível, temperatura, pH e turbidez. O sistema utiliza um microcontrolador ESP32 e comunica com a plataforma Thinger.io para monitorização remota.

### 2. Estação Meteorológica

Sistema destinado à recolha e monitorização de dados ambientais, como temperatura, humidade e pressão atmosférica. Os dados são disponibilizados localmente e através da plataforma IoT utilizada no projeto.

### 3. Portão Automático

Sistema de controlo de acessos baseado em RFID, permitindo a abertura e fecho automático de um portão através de identificação autorizada.

## Tecnologias Utilizadas

* ESP32
* Arduino Mega
* Arduino IDE
* Thinger.io
* Sensores DHT11, BMP388, DS18B20, pH e Turbidez
* RFID RC522
* Displays OLED e TFT
* Relés, motores e atuadores diversos

## Objetivo

O principal objetivo deste projeto consiste em demonstrar a integração de tecnologias de automação, monitorização e Internet das Coisas (IoT), aplicadas a cenários reais de controlo e supervisão.
